Link to the challenge

"Your mission is to create a stopwatch program. this program should have start, stop, and lap options, and it should write out to a file to be viewed later."

Programmeringsplanering

  • Räkna upp hundradelar. Vid 99 ska sekunder räknas upp med 1 när sekunder är 60 ska minuter räknas upp med 1.
  • Lap-funktion ska spara min, sek och hundradelar
  • Start-funktion ska nollställa allt.
  • Stop-funktion ska bara stanna uppräkningen.
